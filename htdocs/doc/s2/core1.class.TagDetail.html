TagDetail Class

TagDetail Class — A rich structure with lots of information about a Tag.

Parent Class

Child class of Tag.

Members

string name

(Read-only) Textual representation of this tag.

int{} security_counts

(Read-only) How many times this tag has used this security. The keys are which security, one of: public, private, friends, group. The value is the count of times the tag is used on entries with that security level.

string url

(Read-only) URL to view entries with this tag.

int use_count

(Read-only) Count of how many times this tag has been used.

string visibility

(Read-only) The visibility level for this tag. Based on the entries it's used on. Can be one of: public, private, friends, group.
