LJ::Talk::unfreeze_thread

LJ::Talk::unfreeze_thread — unfreezes an entire thread of comments.

Use

LJ::Talk::unfreeze_thread(u, jitemid, jtalkid);

Arguments

  • u. Userid or user object to unfreeze thread from.

  • jitemid. Journal itemid of item to unfreeze comments from.

  • jtalkid. Journal talkid of comment at top of thread to unfreeze.

Source:

cgi-bin/talklib.pl

Returns:

1 on success; undef on error
