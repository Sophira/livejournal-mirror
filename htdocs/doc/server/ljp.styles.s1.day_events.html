DAY_EVENTS

DAY_EVENTS — When events occur on a given day in the view, this variable gets inserted into %%events%% of DAY_PAGE.

View Types:

day

Overrideable:

No; users of the style cannot override this. It may only be defined in the style.

Variable Type

HTML

Properties

eventsA bunch of DAY_EVENT variables for the events that occured on that day, in the order dictated by DAY_SORT_MODE.
