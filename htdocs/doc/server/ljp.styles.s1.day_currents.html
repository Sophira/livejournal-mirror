DAY_CURRENTS

DAY_CURRENTS — When a user has specified "current" information (like current music or current mood), then this variable separates those entries apart from the rest of the journal entry in DAY_EVENT.

View Types:

day

Overrideable:

No; users of the style cannot override this. It may only be defined in the style.

Variable Type

HTML

Properties

currentsThis property is one or more DAY_CURRENT variables joined together, one for each current item.
