Name

LJ::get_effective_user — Given a set of input, will return the effective user to process as.

Use

LJ::get_effective_user(dbs, opts);

Arguments

  • dbs. A master/slave database pair, as generated by LJ::get_dbs.

  • opts. A hash of options to pass.

Info

Is passed a reference to a form hash, a remote hash reference, a reference to an error variable, and a reference to a user hash to possibly fill. Given the form input, it will authenticate and return the user (logged in user, a community, other user) that the remote user requested to do an action with.

Source:

cgi-bin/ljlib.pl

Returns:

The user to process as.
