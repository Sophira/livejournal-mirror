LASTN_CURRENT

LASTN_CURRENT — When a user has specified "current" information (like current music or current mood), then this variable is added to the %%currents%% property of LASTN_CURRENTS.

View Types:

lastn

Overrideable:

No; users of the style cannot override this. It may only be defined in the style.

Variable Type

HTML

Properties

whatSpecifies what this is the current item of. It'll be something like "Mood", "Music", or "Weather". You can change the case with the transforms (%%lc:what%%, etc...)
valueThe actual current whatever.
