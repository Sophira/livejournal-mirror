LJ::acct_code_encode

LJ::acct_code_encode — Given an account ID integer and a 5 digit auth code, returns a 12 digit account code.

Use

LJ::acct_code_encode(acid, auth);

Arguments

  • acid. account ID, a 4 byte unsigned integer

  • auth. 5 random characters from base 30 alphabet.

Source:

cgi-bin/ljlib.pl

Returns:

12 digit account code.
