LJ::make_dbs_from_arg

LJ::make_dbs_from_arg — Convert unknown arg to a dbset.

Use

LJ::make_dbs_from_arg(something);

Arguments

  • something. Either a db handle or a dbset.

Info

Functions use this to let their callers use either db handles or dbsets. If argument is a single handle, turns it into a dbset. If already a dbset, just returns it unchanged.

Source:

cgi-bin/ljlib.pl

Returns:

A dbset.
