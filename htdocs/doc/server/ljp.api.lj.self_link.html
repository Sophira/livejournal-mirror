LJ::self_link

Name

LJ::self_link — Takes the URI of the current page, and adds the current form data to the url, then adds any additional data to the url.

Use

LJ::self_link(form, newvars);
form;
newvars;

Arguments

  • form. A hashref of the form information from the page.

  • newvars. A hashref of information to add/override to the link.

Source:

cgi-bin/ljlib.pl

Returns:

scalar; the full url
