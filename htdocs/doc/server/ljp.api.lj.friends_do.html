LJ::friends_do

LJ::friends_do — Runs given sql, then deletes the given userid's friends from memcache

Use

LJ::friends_do(uuserid, sql, args);

Arguments

  • uuserid. a userid or u object

  • sql. sql to run via $dbh->do()

  • args. a list of arguments to pass use via: $dbh->do($sql, undef, @args)

Source:

cgi-bin/ljlib.pl

Returns:

results of $dbh->do()
