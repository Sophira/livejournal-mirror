LJ::send_statserv

Name

LJ::send_statserv — Sends UDP packet of info to the statistics server.

Use

LJ::send_statserv(cachename, ip, type, url);

Arguments

  • cachename. The name to cache this client under. This is can be the logged in username, the value of a guest cookie, or simply "ip" to indicate a cookie-less client.

  • ip. The dotted quad representing the client's IP address.

  • type. What type of client this is. "user", "guest" or "ip".

  • url. An optional URL of what the client hit.

Source:

cgi-bin/ljlib.pl

Returns:

Nothing.
