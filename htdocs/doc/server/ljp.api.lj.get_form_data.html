LJ::get_form_data

Name

LJ::get_form_data — Loads a hashref with form data from a GET or POST request.

Use

LJ::get_form_data(hashref, type);
hashref;
type;

Arguments

  • hashref. Hashref to populate with form data.

  • type. If "GET", will ignore POST data.
