LJ::html_hidden

Name

LJ::html_hidden — Makes the HTML for a hidden form element

Use

LJ::html_hidden(name, val);

Arguments

  • name. Name of form element (will be HTML escaped)

  • val. Value of form element (will be HTML escaped)

Source:

cgi-bin/ljlib.pl

Returns:

HTML
