LJ::get_cluster_description

LJ::get_cluster_description — Get descriptive text for a cluster id.

Use

LJ::get_cluster_description(clusterid, bold);

Arguments

  • clusterid. id of cluster to get description of

  • bold. 1 == bold cluster name and subcluster id, else don't

Source:

cgi-bin/ljlib.pl

Returns:

string representing the cluster description
