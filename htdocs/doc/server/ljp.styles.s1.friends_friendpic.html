FRIENDS_FRIENDPIC

FRIENDS_FRIENDPIC — Creates the IMG tag needed to show your friend's picture on the friends view page.

View Types:

friends

Overrideable:

No; users of the style cannot override this. It may only be defined in the style.

Variable Type

HTML

Properties

srcThe IMG SRC= attribute.
widthThe IMG WIDTH= attribute.
heightThe IMG HEIGHT= attribute.
