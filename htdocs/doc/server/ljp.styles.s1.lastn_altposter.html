LASTN_ALTPOSTER

LASTN_ALTPOSTER — When a user other than the journal owner posts a top-level journal entry in a journal, this variable is inserted into the %%altposter%% property of LASTN_EVENT.

View Types:

lastn

Overrideable:

No; users of the style cannot override this. It may only be defined in the style.

Variable Type

HTML

Properties

posterThe username that's posting.
ownerThe username that owns the journal.
picIf the user posting has a picture, the contents of LASTN_ALTPOSTER_PIC go here.
