LASTN_SKIP_BACKWARD

LASTN_SKIP_BACKWARD — The HTML to make the link to the next (less recent) page of the lastn view... going back in time.

View Types:

lastn

Overrideable:

No; users of the style cannot override this. It may only be defined in the style.

Variable Type

HTML

Properties

numitemsThe number of items being displayed on the next page.
urlThe URL of the page. (the HREF of your A tag)
