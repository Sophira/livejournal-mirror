FRIENDS_SUBJECT

FRIENDS_SUBJECT — When a journal entry has a subject, this variable is inserted into the %%subject%% property of FRIENDS_EVENT.

View Types:

friends

Overrideable:

No; users of the style cannot override this. It may only be defined in the style.

Variable Type

HTML

Properties

subjectThe journal entry's subject.
