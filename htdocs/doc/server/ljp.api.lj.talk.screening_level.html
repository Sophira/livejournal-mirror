LJ::Talk::screening_level

LJ::Talk::screening_level — Determines the screening level of a particular post given the relevent information.

Use

LJ::Talk::screening_level(journalu, jitemid);

Arguments

  • journalu. User object of the journal the post is in.

  • jitemid. Itemid of the post.

Source:

cgi-bin/talklib.pl

Returns:

Single character that indicates the screening level. Undef means don't screen
