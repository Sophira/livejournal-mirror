LJ::dbs_selectrow_hashref

LJ::dbs_selectrow_hashref — Like DBI's selectrow_hashref, but working on a $dbs preferring the slave.

Use

LJ::dbs_selectrow_hashref(dbs, query);

Arguments

  • dbs. A master/slave database pair, as generated by LJ::get_dbs.

  • query. The select query to run.

Info

Given a dbset and a query, will try to query the slave first. Falls back to master if not in slave yet. See also LJ::dbs_selectrow_array.

Source:

cgi-bin/ljlib.pl

Returns:

Hashref, or undef if no row found in either slave or master.
