LJ::debug

LJ::debug — When $LJ::DEBUG is set, logs the given message to the Apache error log. Or, if $LJ::DEBUG is 2, then prints to STDOUT.

Use

LJ::debug(message);

Arguments

  • message. Message to log.

Source:

cgi-bin/ljlib.pl

Returns:

1 if logging disabled, 0 on failure to open log, 1 otherwise
