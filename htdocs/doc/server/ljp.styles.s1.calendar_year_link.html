CALENDAR_YEAR_LINK

CALENDAR_YEAR_LINK — A link to view a different year in the calendar.

View Types:

calendar

Overrideable:

No; users of the style cannot override this. It may only be defined in the style.

Variable Type

HTML

Properties

urlThe URL to go to the other year.
yyyyThe 4 digit year this link goes to.
yyThe 2 digit year this link goes to.
