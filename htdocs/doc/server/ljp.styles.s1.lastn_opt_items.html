LASTN_OPT_ITEMS

LASTN_OPT_ITEMS — This is an integer value between 1 and 50 that sets how many items are shown in the "events" property of the LASTN_PAGE variable.

View Types:

lastn

Overrideable:

Yes; users of this style may override this

Variable Type

integer

Default Value

20
