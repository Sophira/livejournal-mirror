»1. Install MySQL

LiveJournal requires a MySQL install of at least version 3.23.x, but highly recommends installing the latest stable release that is available, expecially if you're going to need clustering and replication abilities.

MySQL offers binaries for many operating aystems, including Win32 and several different “flavors” of Unix. Installations on any of these systems should be fairly easy, as most offer a simple installation program that guides you through setup.

Debian

The MySQL server package for Debian is mysql-server.
