FRIENDS_RANGE_MOSTRECENT

FRIENDS_RANGE_MOSTRECENT — This should be the text that says the user is viewing the top (most recent) entries, and there are %%numitems%% of them.

View Types:

friends

Overrideable:

No; users of the style cannot override this. It may only be defined in the style.

Variable Type

HTML

Properties

numitemsThe number of items being displayed.
