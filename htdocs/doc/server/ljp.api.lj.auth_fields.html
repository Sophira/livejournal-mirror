Name

LJ::auth_fields — Makes a login form.

Use

LJ::auth_fields(form, opts);

Arguments

  • form. The hash of form information, which is used to determine whether to get the current login info and display a concise form, or to display a login form.

  • opts. hashref containing 'user' key to force (finds/makes the hpassword)

Info

Returns a form for either submitting username/password to a script or entering a new username/password.

Source:

cgi-bin/ljlib.pl

Returns:

The built form
