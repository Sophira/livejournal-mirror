LJ::is_utf8

LJ::is_utf8 — check text for UTF-8 validity

Use

LJ::is_utf8(text);

Arguments

  • text. text to check for UTF-8 validity

Source:

cgi-bin/ljlib.pl

Returns:

1 if text is a valid UTF-8 stream, 0 otherwise.
