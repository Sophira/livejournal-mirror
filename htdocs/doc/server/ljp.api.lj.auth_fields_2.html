Name

LJ::auth_fields_2 — Makes a login form.

Use

LJ::auth_fields_2(form, opts);

Arguments

  • form. Form results from the previous page.

  • opts. Journal/password options for changing the login box.

Info

Like LJ::auth_fields, with a lot more functionality. Creates the HTML for a login box if user not logged in. Creates a drop-down selection box of possible journals to switch to if user is logged in.

Source:

cgi-bin/ljlib.pl

Returns:

The resultant HTML form box.
