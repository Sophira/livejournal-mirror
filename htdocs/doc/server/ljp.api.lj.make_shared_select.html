Name

LJ::make_shared_select — Creates a list of shared journals a user has access to for insertion into a drop-down menu.

Use

LJ::make_shared_select(u, form, opts);

Arguments

  • u. A hashref of a row from the user table, sometimes called a "user object". Although only initially a row from the user table, gets a lot of tacked onto it from various other LJ API functions.

  • form. The form hash from the previous page.

  • opts. A hash of options to change the types of selections shown.

Source:

cgi-bin/ljlib.pl

Returns:

The HTML for the options menu.
