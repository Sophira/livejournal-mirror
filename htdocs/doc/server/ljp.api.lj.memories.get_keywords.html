LJ::Memories::get_keywords

LJ::Memories::get_keywords — Retrieves keyword/keyids without big joins, returns a hashref.

Use

LJ::Memories::get_keywords(uobj);

Arguments

  • uobj. User object to get keyword pairs for.

Source:

cgi-bin/ljmemories.pl

Returns:

Hashref; { keywordid => keyword }
