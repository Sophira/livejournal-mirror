FRIENDS_WEBSITE

FRIENDS_WEBSITE — When the user has specified their website URL, this variable goes into the %%website%% property of FRIENDS_PAGE.

View Types:

friends

Overrideable:

Yes; users of this style may override this

Variable Type

HTML

Properties

urlThe URL of the user's website.
nameThe name of the user's website, or "my website" if they haven't specified it.
