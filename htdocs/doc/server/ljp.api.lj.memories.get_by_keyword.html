LJ::Memories::get_by_keyword

LJ::Memories::get_by_keyword — Get memories given a user and a keyword/keyword id.

Use

LJ::Memories::get_by_keyword(uuobj, kwoid, opts);

Arguments

  • uuobj. User id or user object to get memories for.

  • kwoid. Keyword (string) or keyword id (number) to get memories for.

  • opts. Hashref of extra options to pass through to memory getter. Suggested options

Source:

cgi-bin/ljmemories.pl

Returns:

Hashref of memories with keys being memid; undef on error.
