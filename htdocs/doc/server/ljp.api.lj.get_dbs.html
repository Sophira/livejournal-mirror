LJ::get_dbs

LJ::get_dbs — Returns a set of database handles to master and a slave, if this site is using slave databases. Only use this once per connection and pass around the same $dbs, since this function calls LJ::get_dbh which uses cached connections, but validates the connection is still live.

Use

LJ::get_dbs

Source:

cgi-bin/ljlib.pl

Returns:

$dbs (see LJ::make_dbs)
