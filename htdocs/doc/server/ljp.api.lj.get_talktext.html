LJ::get_talktext

Name

LJ::get_talktext — Efficiently retrieves a large number of comments, trying first slave database servers for recent items, then the master in cases of old items the slaves have already disposed of. See also: LJ::get_logtext.

Use

LJ::get_talktext(dbs, opts, talkid);

Arguments

  • dbs. A master/slave database pair, as generated by LJ::get_dbs.

  • opts. Optional hashref of flags. Currently supported key: 'onlysubjects', which won't return body text: $body will be undef.

  • talkid. List of talkids to retrieve the subject & text for.

Source:

cgi-bin/ljlib.pl

Returns:

hashref with keys being talkids, values being [ $subject, $body ]
