DAY_HEAD

DAY_HEAD — Any arbitrary HTML you want placed in the <HEAD> of the page. This is a good place to put CSS or META tags. If you want to add the same tags to all your pages, use GLOBAL_HEAD.

View Types:

day

Overrideable:

Only; Only users of this style may override this, it cannot be defined in a style.

Variable Type

HTML
