LJ::load_user_theme

Name

LJ::load_user_theme — Populates a variable hash with color theme data.

Use

LJ::load_user_theme(user, u, vars);

Arguments

  • user. The username to search for data with.

  • u. A hashref of a row from the user table, sometimes called a "user object". Although only initially a row from the user table, gets a lot of tacked onto it from various other LJ API functions.

  • vars. A hashref to fill with color data. Adds keys "color-$coltype" with values $color.

Source:

cgi-bin/ljlib.pl

Returns:

Nothing. Modifies a hash reference.
