Making Journal Styles

Styles are the part of LiveJournal that allows users to customize the look of their accounts.

Currently there are two types of style interfaces available to use, the older ‘S1’ which is very linear and dumb, and the newer ‘S2’ which features its own sort of programming language. Read on for more information on S1, or visit The S2 Manual» for more information on S2.
