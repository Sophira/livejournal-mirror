LJ::load_remote

Name

LJ::load_remote — Given a partial remote user hashref (from LJ::get_remote), loads in the rest, unless it's already loaded.

Use

LJ::load_remote(dbarg, remote);

Arguments

  • dbarg. A master database handle ($dbh) or a master/slave set of database handles ($dbs) as generated by LJ::get_dbs.

  • remote. Hashref containing 'user' and 'userid' keys at least. This hashref will be populated with the rest of the 'user' table data. If undef, does nothing.
