LJ::load_style_fast

Name

LJ::load_style_fast — Loads a style, and does minimal caching (data sticks for 60 seconds).

Use

LJ::load_style_fast(styleid, dataref, typeref, nocache);

Arguments

  • styleid. Numeric, primary key.

  • dataref. Dataref to store data in.

  • typeref. Optional dataref to store the style tyep in (undef for none).

  • nocache. Flag to say don't cache.

Source:

cgi-bin/ljlib.pl

Returns:

Nothing. Modifies a data reference.
