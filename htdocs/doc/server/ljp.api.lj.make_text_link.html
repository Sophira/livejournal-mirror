LJ::make_text_link

Name

LJ::make_text_link — The most pathetic function of them all. AOL's shitty mail reader interprets all incoming mail as HTML formatted, even if the content type says otherwise. And AOL users are all too often confused by a a URL that isn't clickable, so to make it easier on them (*sigh*) this function takes a URL and an email address, and if the address is @aol.com, then this function wraps the URL in an anchor tag to its own address. I'm sorry.

Use

LJ::make_text_link(url, email);

Arguments

  • url. URL to return or wrap.

  • email. Email address this is going to. If it's @aol.com, the URL will be wrapped.

Source:

cgi-bin/ljlib.pl

Returns:

the same URL, or the URL wrapped in an anchor tag for AOLers
