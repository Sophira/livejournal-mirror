LJ::make_remote

LJ::make_remote — Returns a minimal user structure ($remote-like) from a username and userid.

Use

LJ::make_remote(user, userid);

Arguments

  • user. Username.

  • userid. User ID.

Source:

cgi-bin/ljlib.pl

Returns:

hashref with 'user' and 'userid' keys, or undef if either argument was bogus (so caller can pass untrusted input)
