LJ::dbs_selectrow_array

LJ::dbs_selectrow_array — Like DBI's selectrow_array, but working on a $dbs preferring the slave.

Use

LJ::dbs_selectrow_array(dbs, query);

Arguments

  • dbs. A master/slave database pair, as generated by LJ::get_dbs.

  • query. The select query to run.

Info

Given a dbset and a query, will try to query the slave first. Falls back to master if not in slave yet. See also LJ::dbs_selectrow_hashref.

Source:

cgi-bin/ljlib.pl

Returns:

In scalar context, the first column selected. In list context, the entire row.
