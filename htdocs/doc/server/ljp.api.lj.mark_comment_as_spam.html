LJ::mark_comment_as_spam

LJ::mark_comment_as_spam — Copies a comment into the global spamreports table

Use

LJ::mark_comment_as_spam(journalu, jtalkid);

Arguments

  • journalu. User object of journal comment was posted in.

  • jtalkid. ID of this comment.

Source:

cgi-bin/talklib.pl

Returns:

1 for success, 0 for failure
