LJ::http_to_time

LJ::http_to_time — Converts HTTP date to Unix time.

Use

LJ::http_to_time(string);

Arguments

  • string. HTTP Date. See RFC 2616 for format.

Info

Wrapper around HTTP::Date::str2time. See also LJ::time_to_http.

Source:

cgi-bin/ljlib.pl

Returns:

integer; Unix time.
